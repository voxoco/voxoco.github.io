HIPAA

HIPPA Compliance is a large undertaking. VOXO stives to make working within the regulations as easy as possible providing solutions that meeting HIPAA specifications.

  • Unique User ID & authentication. Phones identify themselves with the phone number or serial number on the phone.
  • Access Controls. Certain users may have additional privileges beyond making phone calls, the system has several different classes of users.
  • Audit logs. The system tracks call meta data, as well details regarding any administrative activities performed by an authenticated user.
  • Encryption. TLS and or VPNs can be employed between IP Phones and the Communications Manager Software. For data at rest, for example, voicemails, other encryption technologies can be used.
